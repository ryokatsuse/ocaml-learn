## 変数の定義

- 同じ値が複数の箇所にある場合に修正が困難になる。
- そこで変数！

Ocamlでは以下のように書く

```
let 変数名 = 式
```
- 変数を宣言すると型と変数にセットされた値が出力される
- 先頭の文字はアルファベット小文字ではないといけない

例：時給を変数にセットして計算する

```
# let jikyu = 850;;
val jikyu: int = 850

# (25 * jikyu) + (28 * jikyu) + (31 * jikyu) ;;
- : int = 71400
```

## 他の変数との違い

- 変数といっても定数に近いもの
- Cなどの命令型言語の変数は一度値を代入した後で別の値に置き換え可能
- 関数型では基本的に値はずっとそのままである
- プログラミングの見通しをよくする
